**피보나치 수열을 구하는 함수**를 반복문으로 만들 때
재귀함수의 개념인 f(n) = f(n-1) + f(n-2)를 그대로 나타내야 했습니다.
이를 위해 총 세개의 변수를 설정,
그것이 
```
    now = 0
    p = 1
    pp = 0
```
이 부분입니다.
여기서 now는 수열의 해당하는 방, 즉 f(n)을 의미하고
p는 f(n-1), pp는 f(n-2)에 해당합니다.
반복문을 살펴보면
```
    for cnt in range(nbr):
    pp = p
    p = now
    now = pp+p
```
이 구조로 되어있는데 여기서 range(nbr)은 
입력한 숫자만큼 반복문이 동작한다는 것을 의미하며
즉, 수열의 항이 구해진다는 것과 동일합니다.
예를 들어 **nbr = 3**일 때 반복문의 동작을 따라가보면
```
    now  p   pp
    0    1   0   -> 초기값, 즉 0번항
    0    1   1   -> pp = p
    0    0   1   -> p = now
    1    0   1   -> now = pp+p, 1번항
    1    0   0   -> pp = p
    1    1   0   -> p = now
    1    1   0   -> now = pp+p, 2번항
    1    1   1   -> pp = p
    1    1   1   -> p = now
    2    1   1   -> now = pp+p, 3번항
```
으로 0 1 1 2 라는 수열에서 f(n)과 f(n-1), f(n-2)를 구하는 것을 볼 수 있습니다.
또한 시간 복잡도는 n에 비례하며 2^n에 비례하는 재귀함수보다 
훨씬 간단한 것 역시 확인할 수 있습니다.
